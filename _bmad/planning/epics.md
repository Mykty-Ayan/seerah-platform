# Epics: Seerah Platform

## Epic 1: Курс/Серия видео
**Описание:** Управление учебными курсами/сериями видео-лекций по сире.

**Features:**
- Создание курса (название, описание, лектор, категория, превью)
- Добавление видео в курс (порядок, название эпизода)
- Отслеживание прогресса пользователя
- Карточка курса на главной (превью, название, лектор, прогресс)

**User Stories:**
- [ ] Как админ, я хочу создать курс с названием, описанием и лектором
- [ ] Как админ, я хочу добавить видео в курс с порядковым номером
- [ ] Как пользователь, я хочу видеть прогресс изучения курса (круговой прогресс-бар)
- [ ] Как пользователь, я хочу начать/продолжить курс с кнопки "Бастау"
- [ ] Как пользователь, я хочу видеть список курсов на главной с превью

**Tech Tasks:**
- [ ] DB: Таблицы `courses`, `course_videos`, `user_course_progress`
- [ ] API: CRUD для курсов, эндпоинт прогресса
- [ ] Frontend: Компонент карточки курса, круговой прогресс-бар

---

## Epic 2: Каталог видео с фильтрацией
**Описание:** Просмотр и фильтрация видео-курсов по категориям.

**Features:**
- Табы категорий (Барлығы, Құран, Ақида, Фиқһ)
- Карточки курсов в сетке
- Превью, название, лектор, количество видео, прогресс

**User Stories:**
- [ ] Как пользователь, я хочу фильтровать курсы по категориям через табы
- [ ] Как пользователь, я хочу видеть карточку курса с превью, названием, лектором, количеством видео
- [ ] Как пользователь, я хочу видеть прогресс курса (линейный прогресс-бар с процентами)
- [ ] Как админ, я хочу создавать категории (tags/topics)

**Tech Tasks:**
- [ ] DB: Таблица `categories` или `tags`, связь M2M с курсами
- [ ] API: Эндпоинт `/courses?category={id}`
- [ ] Frontend: Компонент табов, сетка карточек

---

## Epic 3: Плеер видео и страница курса
**Описание:** Просмотр видео-лекций внутри курса.

**Features:**
- Страница курса со списком эпизодов
- Видео-плеер (YouTube embed или self-hosted)
- Навигация между эпизодами (next/prev)
- Отметка просмотра (авто или ручная)
- Обновление прогресса

**User Stories:**
- [ ] Как пользователь, я хочу открыть курс и увидеть список всех эпизодов
- [ ] Как пользователь, я хочу смотреть видео в плеере
- [ ] Как пользователь, я хочу переключаться между эпизодами (next/prev)
- [ ] Как пользователь, я хочу, чтобы прогресс обновлялся после просмотра
- [ ] Как пользователь, я хочу видеть, какие эпизоды уже просмотрены

**Tech Tasks:**
- [ ] DB: Таблица `user_video_progress` (user_id, video_id, watched, watched_at)
- [ ] API: Эндпоинт `/courses/{id}/videos`, `/progress/update`
- [ ] Frontend: Страница курса, видео-плеер, список эпизодов с чекбоксами

---

## Epic 4: Личный раздел "Менің иманым"
**Описание:** Персонализированный раздел с моими курсами и прогрессом.

**Features:**
- Мои курсы (начатые, завершённые)
- Общий прогресс (статистика)
- Продолжить просмотр (последний видео)
- Сертификаты/награды (опционально)

**User Stories:**
- [ ] Как пользователь, я хочу видеть список моих курсов в разделе "Менің иманым"
- [ ] Как пользователь, я хочу видеть общий прогресс (сколько курсов завершено)
- [ ] Как пользователь, я хочу продолжить последний просмотренный курс
- [ ] Как пользователь, я хочу видеть историю просмотров

**Tech Tasks:**
- [ ] API: Эндпоинт `/user/courses`, `/user/progress-summary`
- [ ] Frontend: Страница "Мои курсы", компоненты статистики

---

## Epic 5: Админ-панель
**Описание:** Управление контентом (курсы, видео, лекторы, категории).

**Features:**
- Dashboard со статистикой (всего курсов, видео, пользователей)
- CRUD курсов (название, описание, лектор, категория, превью)
- CRUD видео (название, URL, длительность, порядок в курсе)
- CRUD лекторов (имя, фото, био)
- CRUD категорий
- Загрузка превью (обложки курсов)

**User Stories:**
- [ ] Как админ, я хочу видеть dashboard с базовой статистикой
- [ ] Как админ, я хочу создать новый курс с обложкой, описанием, лектором
- [ ] Как админ, я хочу добавить видео в курс с URL (YouTube/embed)
- [ ] Как админ, я хочу создать лектора с фото и био
- [ ] Как админ, я хочу создать категории (Құран, Ақида, Фиқһ, etc.)
- [ ] Как админ, я хочу загрузить обложку для курса

**Tech Tasks:**
- [ ] Auth: JWT/Session для админов
- [ ] DB: Таблицы `admins`, `lecturers`, `media_uploads`
- [ ] API: Все CRUD эндпоинты для админки
- [ ] Frontend Admin: Dashboard, формы создания/редактирования, загрузка файлов
- [ ] Storage: Cloudinary/S3 для обложек

---

## Epic 6: Мультиязычность (i18n)
**Описание:** Поддержка нескольких языков (казахский дефолт, русский, английский, арабский).

**Features:**
- Переключение языка
- Переводы UI (табы, кнопки, навигация)
- Переводы контента (названия курсов, описания)

**User Stories:**
- [ ] Как пользователь, я хочу переключить язык интерфейса
- [ ] Как админ, я хочу добавить перевод для курса/описания на несколько языков
- [ ] Как пользователь, я хочу видеть UI на своём языке

**Tech Tasks:**
- [ ] i18n: React-i18next или аналог
- [ ] DB: Поддержка мультиязычных полей (JSON или отдельные таблицы)
- [ ] API: Endpoint `/locale` или header `Accept-Language`

---

## Epic 7: Bottom Navigation и роутинг
**Описание:** Навигация между разделами приложения (Бас бет, Менің иманым, Баптаулар).

**Features:**
- Bottom navigation bar (3 таба)
- Роутинг между страницами
- Активное состояние текущей страницы

**User Stories:**
- [ ] Как пользователь, я хочу переключаться между разделами через bottom nav
- [ ] Как пользователь, я хочу видеть активный таб подсвеченным
- [ ] Как пользователь, я хочу, чтобы URL обновлялся при навигации

**Tech Tasks:**
- [ ] Frontend: React Router, BottomNavigation компонент
- [ ] UX: Smooth transitions, active state styling
