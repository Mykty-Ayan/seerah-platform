# Product Context: Seerah Platform

## Vision
Платформа для изучения жизнеописания Пророка Мухаммада ﷺ через качественные видео-лекции от уважаемых учёных и лекторов.

## Target Users
1. **Студенты/Ищущие знания** — хотят изучать сиру систематически
2. **Преподаватели** — ищут материалы для уроков
3. **Обычные мусульмане** — хотят узнать больше о Пророке ﷺ

## Problem Statement
- Трудно найти качественные, проверенные материалы по сире в одном месте
- Видео разбросаны по YouTube, Telegram, другим платформам
- Нет структурированного подхода (по темам, хронологии, лекторам)

## Solution
Централизованная платформа с:
- Структурированным каталогом видео-лекций
- Фильтрами по темам, лекторам, сериям
- Удобным плеером и навигацией
- Админ-панелью для кураторов контента

## MVP Scope (Phase 1)
### Public Site
- Главная страница с избранными/новыми видео
- Каталог с поиском и фильтрами
- Страница видео с плеером
- Категории/теги/лекторы

### Admin Panel
- Dashboard со статистикой
- CRUD для видео (название, описание, URL, thumbnail, лектор, категории)
- Управление лекторами
- Управление категориями/тегами
- Загрузка/управление медиа (или интеграция с YouTube/Vimeo)

## Out of Scope (Phase 1)
- Пользовательская авторизация (кроме админов)
- Комментарии/лайки
- Платный доступ/подписки
- Мобильное приложение
- Мультиязычность (только один язык на старте)

## Technical Stack
- **Backend:** Go (chi router) + PostgreSQL
- **Frontend Admin:** React + Vite + TypeScript + Shadcn/ui
- **Public Site:** (TBD — может быть отдельный React app или SSR)
- **Video Hosting:** YouTube embed / Cloudinary / self-hosted (TBD)
- **Deploy:** Railway

## Success Metrics (MVP)
- 100+ видео в каталоге
- 50+ уникальных посетителей в день
- Время на сайте > 5 минут
- Админы могут добавлять видео за < 2 минуты
